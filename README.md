## Featured Projects

### Real-Time Ride Analytics Pipeline
**Azure Event Hubs | Azure Data Factory | ADLS | Databricks | Delta Live Tables | Star Schema**

Built an end-to-end Azure data engineering pipeline that combines real-time ride events with batch reference data. The project ingests streaming data through Event Hubs, batch data through ADF, processes it in Databricks, and creates Silver OBT and Gold star-schema models for analytics.

**Why it matters:** Shows streaming + batch ingestion, medallion architecture, dimensional modeling, CDC/SCD thinking, and BI-ready data design.

🔗 [View Project](https://github.com/NavalDhandha/Real_Time_Ride_Analytics_Pipeline)

---

### Airbnb Analytics Engineering with Snowflake + dbt
**Snowflake | dbt | SQL | Incremental Models | Macros | Bronze/Silver/Gold**

Built a modern analytics engineering pipeline that transforms raw Airbnb-style booking, host, and listing data into trusted reporting models. The project includes incremental dbt models, reusable macros, Bronze/Silver/Gold layers, a Gold OBT, and a fact-style reporting model.

**Why it matters:** Shows warehouse modeling, SQL-based ELT, dbt project structure, reusable transformation logic, and analytics-ready data products.

🔗 [View Project](https://github.com/NavalDhandha/Airbnb_Analytics_Engineering_with-Snowflake-dbt)

---

### Invoice Inventory Intelligence
**Python | Pandas | Scikit-learn | Streamlit | SQLite | Joblib**

Built a machine learning + Streamlit app for finance and supply chain workflows. The app predicts freight costs and flags vendor invoices that may require manual approval.

**Why it matters:** Shows how I connect ML, analytics, and business operations into a user-facing decision-support tool.

🚀 [Live App](https://invoiceinventoryintelligence-naval.streamlit.app/)  
🔗 [View Project](https://github.com/NavalDhandha/Invoice_Inventory_intelligence)

---

### Healthcare Provider Productivity Platform
**Azure Data Factory | Databricks | Medallion Architecture | Healthcare Analytics | KPI Modeling**

Built a healthcare analytics platform that transforms appointment, provider, patient, and location data into productivity insights. The project is designed around Bronze/Silver/Gold layers and provider-level operational KPIs.

**Why it matters:** Shows data modeling for healthcare operations, pipeline design, KPI development, and business-facing analytics.

🔗 [View Project](https://github.com/NavalDhandha/Healthcare_provider_productivity_platform)
